Liste des Commandes de Modération

  • +adminlist - Affiche la liste des personnes ayant la permission Administrateur.
  • +botlist - Affiche la liste de tous les bots présents sur le serveur.
  • +rlist - Affiche la liste des personnes ayant la permission Gérer les rôles.
  • +mute - Rend un membre muet.
  • +unmute - Redonne la parole à un membre.
  • +emoji - Permet de créer un émoji sur le serveur.
  • +hide - Permet de cacher un salon.
  • +unhide - Permet de rendre visible un salon.
  • +lock - Permet de fermer un salon.
  • +unlock - Permet d'ouvrir un salon.
  • +config - Affiche la configuration du bot sur le serveur.
  • +addrole - Permet d'ajouter un rôle à un membre.
  • +delrole - Retire un rôle à un membre.
  • +annonce - Permet de faire une annonce de l'administration.
  • +kick - Expulse un membre du serveur.
  • +ban - Bannit un membre du serveur.
  • +clear - Supprime 1 ou plusieurs messages.
  • +renew - Recrée un salon à l'identique.
  • +slowmode - Met en place un mode lent sur un salon.
  • +voicemute - Mute un membre en vocal.
  • +voiceunmute - Unmute un membre en vocal.
  • +unban - Unban un membre du serveur.
  • +unbanall - Unban tous les membres du serveur.

Liste des Commandes d'Utilitaire

  • +help - Vous permet d'obtenir l'intégralité des commandes du bot et leurs informations.
  • +help all - Vous permet d'obtenir l'intégralité des commandes assignées aux rôles selon leurs niveaux de permissions.
  • +help msg - Affiche la totalité des variables messages.
  • +ping - Permet d'afficher le ping du bot.
  • +avatar [id/mention] - Permet d'obtenir l'avatar d'un membre.
  • +banner [id/mention] - Donne la bannière d'un membre.
  • +serveur info - Permet d'obtenir les informations du serveur.
  • +serveur pic - Donne la photo de profil du serveur.
  • +serveur banner - Donne la bannière du serveur.
  • +roleinfo [role] - Permet d'obtenir des informations sur un rôle.
  • +find [@/ID] - Cherche un membre en vocal sur le serveur.
  • +snipe - Affiche le dernier message supprimé dans le salon.
  • +suggest - Permet de faire une suggestion sur le serveur.
  • +userinfo - Permet d'avoir des informations sur un utilisateur.
  • +bypass - Permet de savoir quel antiraid est bypassé par quel niveau de permission.
  • +vc - Affiche les statistiques du serveur.

Liste des Commandes de Gestion Permission

  • +perm - Affiche les permissions d'un membre sur le serveur.
  • +gestion - Affiche les différents modules de gestion et leurs utilités.
  • +pall - Désactive toutes les permissions du serveur.
  • +padmin - Désactive toutes les permissions administrateur du serveur.
  • +prole - Désactive toutes les permissions rôles du serveur.
  • +pban - Désactive toutes les permissions ban du serveur.
  • +pkick - Désactive toutes les permissions kick du serveur.
  • +pvoc - Désactive toutes les permissions vocales du serveur.
  • +pwebhooks - Désactive toutes les permissions webhooks du serveur.
  • +pviewc - Désactive toutes les permissions de voir les salons du serveur.
  • +pserveur - Désactive toutes les permissions de gérer le serveur.
  • +peveryone - Désactive toutes les permissions Everyone du serveur.
